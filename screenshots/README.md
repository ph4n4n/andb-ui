#### **5. Error States**
- Connection failed
- Loading states
- Empty states

## 📱 **Screenshot Specifications**

### **Resolution**
- **Desktop**: 1920x1080 hoặc 1400x900
- **Tablet**: 1024x768
- **Mobile**: 375x667

### **Format**
- **Primary**: PNG (lossless)
- **Alternative**: JPG (nếu cần compress)

### **Quality**
- High quality (không blur)
- Clear text và icons
- Good contrast
